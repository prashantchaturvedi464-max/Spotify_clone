# Spotify_clone
project__with prashant
🎵 Spotify Clone - Web Music Player

A high-fidelity clone of the Spotify Web Player, focusing on pixel-perfect UI design, audio playback function.

✨ Features
* **Music Playback**: Play, pause, and skip tracks using a custom-built audio controller.
* **Pixel-Perfect UI**: Replicated Spotify's iconic dark theme, sidebar, and navigation.
* **Dynamic Song Cards**: Hover effects and smooth transitions for song/album cards.
* **Responsive Layout**: Designed with CSS Flexbox and Grid to ensure a seamless experience across desktop and mobile devices.
* **Volume & Progress Bar**: Interactive range sliders for controlling volume and tracking song progress.

 🛠️ Tech Stack
* **HTML5**: Structured for accessibility and media embedding.
* **CSS3**: Custom properties (variables), Flexbox, and Grid for the layout.
* **JavaScript (ES6+)**: Logic for handling audio play/pause, track switching, and UI updates.
* 
📂 Project Structure

├── index.html       # Core structure and layout
├── style.css        # Spotify dark theme and responsive styling

🧠 What I Learned
* How to use the **HTML5 Audio API** to control music programmatically.
* Advanced **CSS Grid** techniques to create the complex three-panel Spotify layout (Sidebar, Main Content, Player Bar).
